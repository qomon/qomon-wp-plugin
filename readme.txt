Easily insert your Qomon Form in your site. 

== Description ==

First create a form with https://www.qomon.app.

Then you can add it with a shortcode [qomon-form id=your-form-id] or with the Qomon Form block.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/qomon-form` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add a shortcode, enter [qomon-form id=your-form-id] in it. Or, add a new block, choose Qomon Form, and paste your form id in it.
